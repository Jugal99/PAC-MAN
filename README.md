#PAC-MAN Clone WPF C# (Visual Studio 2019)

To run the application, open the PAC-MAN.sln file in Visual Studio 2019 or any later version of Visual Studio. Click the green "Start" button to launch the application.
This is done using Visual Studio with the WPF App (.NET Framework), employing XAML and C# for development. 
All elements of the game, including Pac-Man's movements, wall collisions, and interactions with ghosts and their movements, is implemented through custom code.
